Minitel - libminicurses

Minitels are French terminals from the 80's, that supported ascii mode over serial link (more). One of their use mode was over a serial link (like a VT100 terminal).

This project is a curses-like library, compatible with the non-extended ascii character set supported by the terminal.

The above shown messenger client was written using this library and the fbchat API.
